# User Directory Mini Project

## Start Here

- Requirements: [requirements.md](requirements.md)
- Starter app: [starter/index.html](starter/index.html)
- Reference solution: [solution/mini-project-solution.js](solution/mini-project-solution.js)

## Starter Files

- `starter/index.html` - UI shell and module entrypoint
- `starter/style.css` - starter styles
- `starter/api.js` - API function stubs
- `starter/ui.js` - UI function stubs
- `starter/main.js` - app flow scaffold

## Run

1. Open `starter/index.html` in a local server (recommended: Live Server).
2. Implement TODOs in `starter/api.js`, `starter/ui.js`, and `starter/main.js`.
3. Compare with `solution/mini-project-solution.js` only after completing the core requirements.
